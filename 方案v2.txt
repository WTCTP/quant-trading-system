# 四、核心3：Black-Litterman（顶级组合方法）

这是机构标准做法。

---

## 公式核心：

\mu_{BL} = \left[(\tau \Sigma)^{-1} + P^T \Omega^{-1} P \right]^{-1} \left[(\tau \Sigma)^{-1} \pi + P^T \Omega^{-1} q \right]

---

## 你不用完全理解，直接用这个简化版👇

---

## Python实现（可用版）

```python
import numpy as np

def black_litterman(cov, market_weights, views, tau=0.05):
    pi = cov @ market_weights   # 市场均衡收益

    # views = 你的alpha预测
    mu = (1 - tau) * pi + tau * views

    return mu
```

---

👉 含义：

* `pi` = 市场隐含收益
* `views` = 你的预测
* `mu` = 最终收益

# 五、核心4：组合优化（真正核心）

---

## 优化目标：

w^* = \arg\max_w \frac{w^T \mu}{\sqrt{w^T \Sigma w}}

---

## 实现（最大Sharpe + 约束）

```python
import cvxpy as cp

def optimize(mu, cov):
    n = len(mu)
    w = cp.Variable(n)

    objective = cp.Maximize(mu @ w - cp.quad_form(w, cov))

    constraints = [
        cp.sum(w) == 1,
        w >= 0,
        w <= 0.3
    ]

    prob = cp.Problem(objective, constraints)
    prob.solve()

    return w.value
```
